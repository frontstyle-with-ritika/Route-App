# 🧭 RouteApp

A simple React Router application built using React, Vite, and React Router DOM. This app demonstrates client-side routing with a clean and responsive interface.

## 🚀 Features

- 🌐 Client-side routing with React Router
- ⚡ Fast navigation without page reloads
- 🧭 Clean URLs for different pages
- 📱 Responsive design
- 🎯 Simple and user-friendly interface

## 🛠 Tech Stack

- React (Vite)
- JavaScript
- React Router DOM
- CSS / Tailwind CSS

## 📦 Installation

```bash
git clone https://github.com/frontstyle-with-ritika/Route-App.git
cd Route-App
npm install
npm run dev
```
